## 🧪 実験デザイン

### コアコンセプト: 「錬金術」と「迷い箱」

#### 1. 錬金術サイクル（行動の即時報酬）

```
不用品発見 → 売却/査定 → その日のうちにご馳走
           ↓
     「片付け = 楽しい」という記憶書き換え
```

**科学的根拠**: オペラント条件づけ
- 行動直後の報酬は、行動を強化する
- 「美味しい寿司」という原始的快楽で、脳内報酬系を活性化
- 「片付け = 辛い」→「片付け = 寿司」というニューロン回路の上書き

#### 2. 迷い箱システム（判断の先送り）

```
捨てるか迷う → 「迷い箱」へ移動 → 30日後に再評価
              ↓
         行動のハードルを下げる
```

**心理学的根拠**: 意思決定疲労の回避
- 人間は1日に下せる決断の回数が限られている
- 「捨てる判断」は高負荷な意思決定
- 「移動する」だけなら意思決定コストが低い

### 実験のルール（両親への説明用）

**ルール①: 捨てない**
- 無理に捨てる判断をしなくていい
- 迷ったら「実験ボックス」（迷い箱）へ移動
- これだけでアプリのデータになる

**ルール②: 探さない**
- JALカードなど、探すのに30分以上かかりそうなものは探さない
- 新規発行・再発行で対応（Delaが手続き）
- 浮いた時間で家族の会話を楽しむ

**ルール③: 即時ご褒美**
- 不用品が見つかったら、その日のうちに価値に変える
- メルカリ、買取業者など
- 売却益（または査定見込み額）で美味しいものを食べる

## 📅 Day 1 キックオフ計画

### タイムライン（想定: 10:00〜19:00）

#### 10:00-10:30 プロジェクト説明会

**スクリプト**:
```
Dela: 「実は、僕が開発しているアプリ『シゴデキ！』について、
       二人にモニター協力をお願いしたいんだ」

Dela: 「高齢の人でも負担なく使えるか、実証実験をしたい。
       だから、いつもの片付けとはルールが違う」

Dela: 「今回のルールは2つだけ：
       ①無理に捨てない（迷い箱に入れるだけでOK）
       ②不用品を美味しいご飯に変える（錬金術！）」

Dela: 「お父さんは『捨てる判断』をしなくていい。
       『実験ボックス』に移動させるだけで、
       僕のアプリの役に立つし、今日の夕飯が豪華になる」
```

**資料**: NotebookLM生成のプレゼンスライド（iPadで表示）

#### 10:30-10:45 ウォームアップ: 財布の整理

**目的**: 
- 最も簡単なタスクで成功体験を作る
- 座ったままできる（身体的負担ゼロ）
- 5分で完了する（心理的ハードル低）

**手順**:
1. 財布を出す
2. レシート・ポイントカードを全部出す
3. 期限切れ → 「実験ボックス」へ
4. よく使う3枚だけ残す
5. 財布を拭く

**報酬**: 「これだけで500円分のデータが取れた！」と称賛

#### 11:00-11:15 JALカードの儀式

**目的**: 「探さないルール」の実演

**手順**:
1. お父さん: 「JALカード、どこいったかなぁ...」
2. Dela: 「探さない！これも実験だから」
3. スマホでJALカード新規申し込み（5分）
4. お父さん: 「えっ、それでいいの？」
5. Dela: 「探す30分 vs 申し込む5分。どっちが賢い？」

**学び**: 時間は有限。探すより新しく始める方が効率的

#### 11:30-12:30 昼食（外食で報酬）

**目的**: 
- 午前中の成功を即座に報酬化
- リラックスした雰囲気で午後の計画を話す

**会話トピック**:
- 「財布スッキリしたでしょ？気持ちいいよね」
- 「午後は何やりたい？テレビ周り？それとも...」
- 親に選択肢を与えることで、主体性を引き出す

#### 13:00-15:00 テレビのIT化

**タスク**:
1. Chromecast（または類似デバイス）設置
2. WiFi設定
3. スペインとのビデオ通話テスト
4. 孫の写真を大画面で見る練習

**狙い**:
- 物理的片付けではなく、「未来への投資」という意味づけ
- スペイン移住後も繋がれるという安心感
- IT=楽しい、という連想づけ

#### 15:00-15:30 休憩 + お茶タイム

**雑談で聞く**:
- アプリの使い心地はどうだった？
- 何が難しかった？
- 次は何をやってみたい？

**データ収集**:
- つまずきポイント
- 言葉の選び方（専門用語が分かりにくいなど）
- UIの改善点

#### 15:30-17:00 小物整理（自由課題）

**選択肢**:
- 玄関の靴整理
- 引き出しの中身チェック
- 古い雑誌・新聞の整理

**ルール**:
- 疲れたらすぐ中断
- 迷ったら全部「実験ボックス」へ
- 無理しない

#### 17:00-17:30 振り返りミーティング

**質問**:
1. 今日一番楽しかったことは？
2. 今日一番大変だったことは？
3. アプリのここが使いにくかった、というのは？
4. 明日もやってみたい？（やりたくない？）

**記録**:
- Delaがメモを取る
- 後日、アプリ改善に反映

#### 18:00-19:00 錬金術の儀式: 豪華な夕食

**場所**: 回転寿司 or 焼肉（親の希望優先）

**乾杯の音頭**:
```
Dela: 「今日の成果：財布1個、JALカード1枚、小物10個。
       これで約5,000円分のデータが取れました！
       実験協力、本当にありがとう。乾杯！」
```

**効果**:
- 物質的報酬（美味しい食事）
- 社会的報酬（感謝される）
- 感情的報酬（息子との時間）
- 認知的報酬（役に立ったという実感）
